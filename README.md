# 🧠 Perceptron Simples em Python

Implementação didática e do zero de um Perceptron clássico (rede neural de uma única camada) desenvolvido em Python para problemas de classificação binária e operações lógicas.

## 🚀 Funcionalidades
- **Treinamento Supervisionado**: O modelo ajusta seus pesos sinápticos iterativamente usando a regra de aprendizado do Perceptron (`fit`).
- **Predição Binária**: Classifica amostras de entrada em duas classes (0 ou 1) (`predict`).
- **Avaliação de Acurácia**: Calcula a métrica de acurácia sobre os dados (`score`).
- **Testes Lógicos Clássicos**: O código inclui a modelagem e solução completa das portas lógicas **AND**, **OR** e **NAND**. O modelo alcança 100% de acerto nestas funções linearmente separáveis.

## 🛠️ Tecnologias Utilizadas
- **Python 3**
- **NumPy** (Para operações matriciais eficientes)

## ⚙️ Como Executar
1. Certifique-se de ter o Python 3 e o NumPy instalados no seu computador.
2. Clone o repositório ou baixe a pasta do projeto.
3. Abra o terminal e navegue até a pasta do projeto.
4. Execute o arquivo principal digitando:
   ```bash
   python main.py
   ```
5. Você verá no terminal o processo de treinamento e a acurácia final para as portas lógicas.
