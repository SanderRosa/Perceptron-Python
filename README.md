# Perceptron Simples

Implementação didática de um Perceptron para classificação binária.

## 📁 Estrutura

```
Perceptron/
├── perceptron.py    ← Classe Perceptron
├── main.py          ← Testes e exemplos
└── README.md        ← Este arquivo
```

## 🚀 Como Usar

```bash
python main.py
```

## 📚 Arquivos

### perceptron.py
Implementação da classe Perceptron com:
- `fit(X, y)` - Treina o modelo
- `predict(X)` - Faz predições
- `score(X, y)` - Calcula acurácia

### main.py
Testa o Perceptron com 3 funções lógicas:
- Função AND
- Função OR
- Função NAND

## ✅ Resultado

```
Função AND   → 100%
Função OR    → 100%
Função NAND  → 100%
```
